    1. Plug the router into the wall.
    2. Reset the router by pressing and holding the reset button on the back for 10 seconds.
    3. Connect the router and one of the computers using an ethernet cable.
    1. Find the IP of your router by clicking the top right option bar -> Wired Connected -> Wired Settings -> the gear under the Wired section. The IP is the Default Route. Copy the IP.
    2. Paste the IP into the url to pull up the configuration page.
    3. Traverse to the setup tab and the Network Setup section of that tab. Under Router IP, change the Local IP Address to the desired group IP.
    4. Click Apply Settings. Since your router's IP has been changed, reset your computer to reset the router IP. Enter the new IP into the url field and return to the setup tab.
    5. Change the Start IP Address to the desired start IP. Change Maximum DHCP Users to the desired number. Click Apply Settings.
    6. To ensure setup has been done properly, open the terminal and type
      $ ifconfig
      and ensure the IP in the enp2s0 section is in the range you specified. If it is, you are done!
